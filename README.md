## 🌟 Project Overview

A modern **Phone Store Web Application** built using React.
This project includes authentication, cart management, wishlist, and dark mode — providing a real-world e-commerce experience.

---

## 🚀 Features

* 🔐 User Authentication (Login & Register)
* 🔑 Forgot Password (OTP simulation)
* 🛒 Add to Cart with Quantity Control
* ❤️ Wishlist (Add/Remove)
* 🌙 Dark Mode Support
* 🔍 Product Search
* 💾 Data stored using LocalStorage
* 💰 Discount Price Display
* 📱 Responsive UI (Mobile Friendly)

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 Bootstrap 5
* 💻 JavaScript (ES6)
* 🗂️ LocalStorage (for data persistence)

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/Rakshitha-K123/phone-store-react.git

# Navigate to project
cd phone-store-react

# Install dependencies
npm install

# Start the app
npm start
```
